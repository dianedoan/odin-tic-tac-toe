# Tic Tac Toe
This [webpage](https://dianedoan.github.io/odin-tic-tac-toe/) is a Tic Tac Toe game that you can play in your browser.

It features an interface that allows players to put in their names, includes a button to start and restart the game, and adds a display element that shows the results once the game ends.

## Description
This project demonstrates the skills learnt from the JavaScript Course in The Odin Project up until the [Factory Functions and the Module Pattern](https://www.theodinproject.com/lessons/node-path-javascript-factory-functions-and-the-module-pattern) topic. 

The main goal of this project is to have as little global code as possible by tucking as much code inside factories as possible. For instance, single instances such as the displayController were wrapped inside an IIFE (module pattern) so it cannot be reused to create additional instances.
